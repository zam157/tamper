一个**简单**的**百度文库提取文字**脚本

### 功能介绍
- 提取文字（F12呼出控制台输入`console.log(getDocContent())`后回车就能获取文本，翻到底才能导出所有的内容）
- 破解本地VIP（能够全文浏览内容，但不能下载文档）（inspired by [@王子周棋洛](https://www.bilibili.com/video/BV1xP411j7t8)）

### 兼容性
**不考虑**兼容性，请使用各大主流浏览器的最新版本
